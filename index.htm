Week 1

  1. I learnt about the foundational structure of a HTML Document.
    For example, a HTML document usually looks like this:
    <html> <-- This indicates that it is a HTML5 Document

    <meta information>
    </head>
    <body>
    <the content of the page>
    </body>
    </html>
